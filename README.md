## Install

Clone the repo and install dependencies:

```bash
git clone --depth 1 --branch main https://github.com/spoekle/CC-Clipsesh.git
cd CC-ClipSesh
npm install
```

## Starting Development

Start the app in the `dev` environment:

```bash
npm start
```
